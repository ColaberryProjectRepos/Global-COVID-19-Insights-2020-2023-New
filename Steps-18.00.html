Adding Data Slicers 18

  1. Start by selecting an empty area on the canvas. Then, from the visualizations pane, choose the Slider option.
  2. In the Fields Pane, locate the Year column in the Calendar Table and drag it into the fields value box.
  3. Populate the following settings for the necessary fields:
Visual for Country:
  • Slicer Settings:
    • Style: Drop Down
    • Slicer Header: Off
    • Values: Font - Arial, Size - 13, Color - #000000, Slider - On
Visual for Date
  • Slicer Settings:
    • Style: Between
    • Slicer Header: Off
    • Values: Font - Arial, Size - 13, Color - #000000, Slider - On
General Properties:
  • Size: Height - 109, Width - 212
  • Position: Horizontal - 0, Vertical - 76
  • Title: On, Font - DIN, Size - 14, Text Color - #FFFFFF, Background Color - #FF9900, Horizontal Alignment - Center
  • Effects: Background - On, Color - #000000, Transparency - 100%, Visual Border - On, Color - #FFC266.
