Creating 'Refreshed Date' Table. 11

                                                                      

  1. First, navigate to the Power Query Editor by selecting 'Transform Data.
  2. click on 'New Source' and choose 'Blank Query.' Now, input the query '= DateTime.LocalNow()' and save it.
  3. Rename the resulting table as 'Data Refresh' and convert it into a table format.
  4. you'll notice a new column; rename it 'Date time.' 
  5. Close & Apply
