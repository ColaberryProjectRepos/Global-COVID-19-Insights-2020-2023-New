Establishment of a 'Calculated Metrics' Table 13

                                                                        

  1. Navigate to the "Report view" within Power BI.
  2. Within the Home Tab, choose "Enter data" and then "Create a new table."
  3. Give the table the name 'Calculated Metrics' Table.
  4. Select "Load."
  5. Right-click on 'Calculated Metrics' and opt for "New Measure."
  6. Copy and paste the DAX formula for the 'New Cases' Measure into the Dax Bar.
  7. Click on the checkmark symbol.
  8. Repeat steps 5-7 until you have created all the calculated metrics using the provided DAX formulas.

New Cases = SUM('WHO-COVID-19-global-data'[New_cases])
New Death = SUM('WHO-COVID-19-global-data'[New_deaths])
YoY% New Cases = VAR CurrentYearValue = SUM('WHO-COVID-19-global-data'[New_cases]) VAR PreviousYearValue = CALCULATE(SUM('WHO-COVID-19-global-data'[New_cases]), SAMEPERIODLASTYEAR('Calendar'[Date]))
RETURN
IF(ISBLANK(PreviousYearValue), BLANK(), (CurrentYearValue - PreviousYearValue) / PreviousYearValue)
YoY% New Death = VAR CurrentYearValue = SUM('WHO-COVID-19-global-data'[New_deaths]) VAR PreviousYearValue = CALCULATE(SUM('WHO-COVID-19-global-data'[New_deaths]), SAMEPERIODLASTYEAR('Calendar'[Date]))
RETURN
IF(ISBLANK(PreviousYearValue), BLANK(), (CurrentYearValue - PreviousYearValue) / PreviousYearValue)
Arrow Cases YoY% = SWITCH(TRUE(),[YoY% New Cases] < 0, UNICHAR(9662),[YoY% New Cases]=0,UNICHAR(9666),UNICHAR(9652) )
Arrow Death YoY% = SWITCH(TRUE(),[YoY% New Death] < 0, UNICHAR (9662),[YoY% New Death]=0,UNICHAR(9666),UNICHAR(9652) )
CFR (Case Fatality Rate) = DIVIDE ([New Death], [New Cases], 0)
