Creating Map Legend 24

In this step, we will explore how to include a map legend. Here are the steps for creating the map legend:
  1. Begin by adjusting the data colors formatting:
    • Activate Diverging.
    • Set the Minimum color to be #00B050(Green).
    • For the Center color, use #FFD700(Yellow).
    • Finally, set the Maximum color to #FF0000(Green).
  2. After completing the color formatting, capture a screenshot of the rectangular box and save it.
These steps will help you create and format a legend for your map.
Adding Image and Label
