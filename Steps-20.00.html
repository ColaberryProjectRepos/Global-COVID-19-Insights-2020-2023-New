Creating Yearly analysis chart Using Line and Clustered Column Chart 20

  1. Click on an empty area within the canvas, and from the visualizations pane, choose the Line and Clustered Column Chart option. 
  2. Drag the Year column from the Calendar table to the X-axis. Populate the following details for the relevant fields:
Visual:
  • X-axis: On, Type - Continuous, Values: Font - Arial, Size - 11, Color - #000000, Title - Off
  • Y-axis: On, Values: Font - Arial, Size - 11, Color - #000000, Title - Off
  • Secondary Y-axis: On, Font - Arial, Size - 11, Color - #000000, Title - Off
  • Legend: On, Position - Top left, Style - Marker (Circle Default), Match Line Color - On, Font - Arial, Size - 12, Color - #000000
  • Gridlines: Horizontal - On, Style - Solid, Color - #FFFFFF, Vertical - Off, Zoom Slider - Off
  • Columns: Color - #118DFF
  • Lines: Series - All, Shape - Solid, Join Type - Round, Colors (YoY% New Cases - 6B007B, YoY% New Death - #44D3E6, Rev YoY% - #67002E, Sales YoY% - #9023DE, YoY% - #9023DE), Markers - Off, Date Labels - Off, Series Labels - Off
General:
  • Properties: Size (Horizontal - 272, Vertical - 501), Position (Horizontal - 192, Vertical - 148), Title - On (Text - "Year", Font - Arial, Size - 12, Color - #FFFFFF, Background Color - #FF9900, Horizontal Alignment - Center)
  • Effects: Background - On, Color - #000000, Transparency - 100%, Visual Border - On, Color - #FFC266, Tooltips - On
