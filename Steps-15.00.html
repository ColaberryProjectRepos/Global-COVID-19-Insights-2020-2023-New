Incorporating the Colaberry and COVID19 Logos into the dashboard 15

  1. Access the "Home" tab on the Power BI ribbon and select the "Image" icon to insert an image onto your report canvas.
  2. Choose the Logo Image: A dialog box will appear, requesting you to pick an image. Browse your computer and select the Colaberry logo image from the folder you downloaded earlier.
  3. Once the logo is chosen, you'll find the "Format" tab within the "Visualizations" pane on the right. Click on General and input the following values:
  4. For the Colaberry Logo:
    • Size: Height = 33, Width = 136
    • Position: Horizontal = 0, Vertical = 0
  5. For the COVID19 Logo:
    • Size: Height = 83, Width = 113
    • Position: Horizontal = 10, Vertical = 31
