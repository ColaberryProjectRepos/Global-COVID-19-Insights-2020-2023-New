Creating a Map 23

Here are the steps:
  1. Begin by adding a Filled Map visualization.
  2. Then, follow these field assignments:
    • Place the "Country" column in to the "Location" field box.
    • Add "New Cases" from the Calculated Metrics table to the "Color Saturation" field box.
    • Include "New Cases, New Death YoY% " and "CFR" from the Calculated Metrics table in the "Tooltips" field box.
    • Similarly, add "New Cases" from the Calculated Metrics table to the "Tooltips" field box.
  3. Set the Default color to "Show - On," and adjust the Color and Border color to be 40% lighter than black. Set the Border thickness to 2.
  4. Enable Selection zoom.
  5. Add a title and ensure that the same formatting style is applied throughout the dashboard (refer to step 4).
  6. Turn on the border and set its color to be 30% darker than white.
  7. Enable Tooltips.
These steps will help you create a filled map to visualize current virus spread globally.
